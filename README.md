## 2. Resumo do Projeto

- Cookenu é um projeto de revisão do módulo de react com APIs, design systems e routes

- A aplicação consiste em um site de receitas com as seguintes telas:
  - Login
  - Cadastro
  - Lista de Receitas
  - Detalhes da Receita
  - Adicionar Receita

## 3. Instalação

- Node.js
- npm

Para instalar as dependências no diretório do seu projeto, execute o seguinte comando:

- npm install

## 4. Iniciando

Para iniciar o aplicativo, execute o seguinte comando:

- npm run start

## 5. Tecnologias e Ferramentas Utilizadas

Este site foi construído com as seguintes tecnologias:

- React
- React-Hooks
- React-Router-Dom
- JavaScript
- Styled-Components
- Chakra ui
- Axios
- API-Rest
- React-Toastify
- LocalStorage
